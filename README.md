## Worker_Management

#### 1.프로젝트 소개
근로자 등록책을 C++로 구현해봄
  
#### 2.기능 설명
  0. 근로자 등록 - 정규직, 일용직, 파트타임으로 나누어 등록
  1. 조회 - 선택조회(조회하고자 하는 사람의 이름을 통해 선택 조회), 전체조회
  2. 편집 - 편집하고자 하는 사람의 이름을 적어 편집 항목 선택 후 저장
  3. 삭제 - 선택 삭제(삭제하고자 하는 사람의 이름을 통해 해당 사람만 삭제),전체 삭제
  4. txt - 현재 저장되어있는 정보를 저장, 저장되어있는 txt 불러오기 기능
  5. 종료
